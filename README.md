# Fort VPN Scripts

## Requisitos

Instalação e configuração do Fortclient VPN usando OpenFortVPN

### Instalação do OpenFortVPN

- [FortclientVPN](https://github.com/gabrielDpadua21/openfortivpn)


## Configuração

```bash
bash config.sh
```

## Instalação

```bash
bash install.sh
```

## usar o comando abaixo para iniciar o VPN

```bash
fortvpn
```
